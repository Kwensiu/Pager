Project Named Pager

An Electron desktop application built with React and TypeScript. It provides a sidebar interface for the centralized management of and access to multiple websites within a single window, designed to enhance productivity by organizing web resources.

Pager use [Shadcn UI](https://ui.shadcn.com/docs) and [Radix UI](https://www.radix-ui.com/).
