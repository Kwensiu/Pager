# 快速开始指南

## 5分钟上手 Pager

### 第一步：启动应用程序

1. 安装并启动 Pager
2. 您将看到左侧的侧边栏和右侧的主窗口
3. 侧边栏默认包含一些示例分组

### 第二步：添加您的第一个网站

1. 点击侧边栏底部的 "+" 按钮
2. 选择"添加网站"
3. 在弹出的对话框中：
   - 输入网站名称（如"ChatGPT"）
   - 输入网站 URL（如"https://chat.openai.com"）
   - 可选：添加描述
4. 点击"保存"

### 第三步：组织您的网站

1. **创建分类**：
   - 右键点击侧边栏空白处
   - 选择"添加分类"
   - 输入分类名称（如"工作"）

2. **创建分组**：
   - 右键点击分类
   - 选择"添加分组"
   - 输入分组名称（如"AI工具"）

3. **移动网站到分组**：
   - 拖拽网站到目标分组
   - 或者右键点击网站 → "移动到" → 选择分组

### 第四步：浏览网站

1. 点击侧边栏中的网站
2. 网站将在主窗口的新标签页中打开
3. 使用标签页顶部的导航按钮：
   - 前进/后退
   - 刷新
   - 主页

### 第五步：基本设置

1. 点击右上角的设置图标（齿轮图标）
2. 配置基本设置：
   - **外观**：选择浅色或深色主题
   - **语言**：选择界面语言
   - **浏览器**：启用 JavaScript 和弹出窗口

## 常用操作

### 添加多个网站

1. 重复"添加网站"步骤
2. 或者批量导入（未来版本支持）

### 编辑网站信息

1. 右键点击网站
2. 选择"编辑"
3. 修改名称、URL 或刷新图标
4. 点击"保存"

### 删除网站或分组

1. 右键点击项目
2. 选择"删除"
3. 确认删除操作

### 搜索网站

1. 使用侧边栏顶部的搜索框
2. 输入网站名称或 URL 的一部分
3. 搜索结果将实时显示

## 快捷键参考

### 导航快捷键

- **Ctrl/Cmd + T**：新建标签页
- **Ctrl/Cmd + W**：关闭当前标签页
- **Ctrl/Cmd + Tab**：切换到下一个标签页
- **Ctrl/Cmd + Shift + Tab**：切换到上一个标签页

### 侧边栏快捷键

- **Alt/Cmd + Shift + S**：显示/隐藏侧边栏
- **Alt/Cmd + 数字键**：切换到对应位置的网站

### 窗口快捷键

- **Alt/Cmd + Shift + T**：切换窗口置顶
- **Alt/Cmd + Shift + M**：切换小窗模式
- **F11**：全屏模式

## 实用技巧

### 技巧1：快速添加常用网站

1. 将浏览器书签导出为 HTML 文件
2. 未来版本将支持书签导入功能

### 技巧2：使用分组提高效率

- 按用途分组：工作、娱乐、学习
- 按项目分组：项目A、项目B
- 按频率分组：每天使用、偶尔使用

### 技巧3：利用侧边栏折叠

- 点击侧边栏顶部的折叠按钮
- 折叠后只显示网站图标
- 适合小屏幕或多任务场景

### 技巧4：备份您的数据

1. 定期导出配置
2. 备份用户数据目录
3. 使用数据同步功能
