# 功能详解

## 侧边栏管理

### 分类和分组系统

Pager 使用两级组织结构来管理网站：

**一级分类**

- 用于组织不同类型的网站（如"工作"、"娱乐"、"学习"）
- 每个分类可以包含多个二级分组
- 分类在侧边栏中显示为可折叠的顶级项目

**二级分组**

- 在分类下组织相关网站
- 例如在"工作"分类下可以有"开发工具"、"项目管理"等分组
- 分组可以拖拽排序

**网站项目**

- 实际访问的网站
- 包含网站名称、URL、图标和可选描述
- 支持拖拽到不同分组

### 拖拽排序

- **分类排序**：拖拽分类标题调整顺序
- **分组排序**：在分类内拖拽分组调整顺序
- **网站排序**：在分组内拖拽网站调整顺序
- **跨分组移动**：将网站拖拽到其他分组

### 右键菜单

- **分类右键**：添加分组、重命名、删除
- **分组右键**：添加网站、重命名、删除
- **网站右键**：编辑、删除、在新标签页打开

## 网站浏览功能

### 多标签页支持

- 每个网站在新标签页中打开
- 标签页可以拖拽重新排序
- 支持标签页预览（鼠标悬停）
- 关闭标签页的快捷键：Ctrl/Cmd + W

### 网站图标管理

- **自动获取**：添加网站时自动从 URL 获取 favicon
- **手动刷新**：编辑网站时点击图标刷新
- **默认图标**：无法获取图标时使用默认图标
- **图标缓存**：图标本地缓存，减少网络请求

### 导航工具栏

- **前进/后退**：浏览历史导航
- **刷新**：重新加载当前页面
- **主页**：返回网站首页
- **地址栏**：显示和编辑当前 URL

## 设置与配置

### 外观设置

- **主题模式**
  - 浅色主题：明亮界面
  - 深色主题：暗色界面，减少眼睛疲劳
  - 跟随系统：自动匹配系统主题

- **界面语言**
  - 中文：简体中文界面
  - 英文：英语界面

- **侧边栏显示模式**
  - 显示所有网站：折叠时显示所有网站图标
  - 仅显示展开的分组：折叠时只显示展开状态的分组

### 浏览器设置

- **启用 JavaScript**：控制是否允许网页执行 JavaScript
- **允许弹出窗口**：控制是否允许网页打开新窗口
- **保存会话数据**：保存登录状态和浏览历史
- **退出时清除缓存**：关闭应用时清理临时文件

### 数据管理

- **重置为默认设置**：恢复所有设置为默认值
- **清除所有数据**：删除所有网站、分组和设置
- **导出配置**：备份当前配置到文件
- **导入配置**：从文件恢复配置

## 扩展管理

### 扩展支持

Pager 支持标准的 Chrome 扩展格式，包括：

- **内容脚本**：修改网页内容的脚本
- **后台脚本**：在后台运行的脚本
- **用户界面**：浏览器操作、弹出窗口等
- **Tampermonkey 脚本**：通过 Tampermonkey 扩展支持

### 扩展安装方式

1. **文件夹安装**：选择包含 `manifest.json` 的扩展文件夹
2. **ZIP 文件安装**：选择扩展的 ZIP 压缩包
3. **CRX 文件安装**：选择 Chrome 扩展的 CRX 文件

### 扩展隔离级别

- **严格隔离**：完全独立的进程和存储
- **标准隔离**：适度的隔离，平衡安全性和兼容性
- **宽松隔离**：最小隔离，最大化兼容性
- **无隔离**：不进行隔离（仅用于调试）

### 权限管理

- **查看权限**：显示扩展请求的所有权限
- **单独控制**：为每个权限单独授权或拒绝
- **权限统计**：显示高风险权限数量
- **权限日志**：记录权限使用情况

## 增强功能

### 浏览器指纹伪装

**功能目的**：防止网站通过浏览器指纹追踪用户

**伪装模式**

- **基础模式**：修改基本指纹信息（用户代理、屏幕分辨率等）
- **平衡模式**：修改更多指纹信息，保持较好的网站兼容性
- **高级模式**：全面修改指纹信息，提供最大隐私保护

**网站特定设置**：可以为特定网站启用或禁用指纹伪装

### 全局快捷键

- **窗口置顶**：Alt/Cmd + Shift + T
- **小窗模式**：Alt/Cmd + Shift + M
- **显示/隐藏侧边栏**：Alt/Cmd + Shift + S
- **新建标签页**：Ctrl/Cmd + T
- **关闭标签页**：Ctrl/Cmd + W

### 系统托盘

- **托盘图标**：在系统托盘显示 Pager 图标
- **快捷菜单**：右键点击托盘图标显示菜单
- **通知**：显示应用程序通知
- **最小化到托盘**：关闭窗口时最小化到托盘

### 内存优化

**自动清理机制**

- **清理间隔**：设置自动清理的时间间隔（分钟）
- **最大不活跃时间**：网站不活跃多长时间后清理
- **内存阈值**：当内存使用超过阈值时触发清理

**清理效果**

- 释放不活跃网站的内存
- 保持应用程序响应速度
- 减少系统资源占用

### 数据同步

**配置同步**

- **导出配置**：将当前设置导出为 JSON 文件
- **导入配置**：从 JSON 文件恢复设置
- **自动同步**：定期备份配置到指定位置

**Cookie 同步**

- **导出 Cookie**：导出所有网站的 Cookie 数据
- **导入 Cookie**：从文件导入 Cookie 数据
- **会话保持**：保持登录状态

### 自动启动

- **开机启动**：系统启动时自动运行 Pager
- **启动参数**：可以配置启动时的参数
- **最小化启动**：启动时最小化到托盘

### JS 代码注入

- **自定义脚本**：向特定网站注入 JavaScript 代码
- **脚本管理**：管理多个注入脚本
- **条件触发**：基于 URL 模式触发脚本

### 代理支持

- **代理配置**：设置 HTTP/HTTPS 代理
- **代理规则**：基于 URL 模式的代理规则
- **连接测试**：测试代理连接是否正常

## 窗口管理

### 窗口模式

- **正常模式**：标准窗口大小
- **小窗模式**：紧凑窗口，适合多任务处理
- **全屏模式**：全屏显示当前网站

### 窗口操作

- **置顶窗口**：窗口始终显示在最前面
- **边缘吸附**：窗口靠近屏幕边缘时自动吸附
- **窗口分割**：支持多窗口并排显示

### 窗口状态

- **最小化**：最小化到任务栏或托盘
- **最大化**：最大化窗口
- **恢复**：恢复到之前的大小和位置

## 版本管理

### 自动更新

- **检查更新**：自动检查新版本
- **下载更新**：自动下载更新包
- **安装更新**：提示用户安装更新

### 版本信息

- **应用程序版本**：显示当前版本号
- **Electron 版本**：显示底层框架版本
- **Chromium 版本**：显示浏览器引擎版本
- **Node.js 版本**：显示运行时版本

## 开发者工具

### 调试功能

- **开发者工具**：打开 Chromium 开发者工具
- **控制台日志**：查看应用程序日志
- **网络监控**：监控网络请求

### 诊断工具

- **内存使用**：显示内存使用情况
- **进程信息**：显示所有进程状态
- **性能分析**：性能分析工具

## 隐私与安全

### 数据保护

- **本地存储**：所有数据存储在本地
- **加密选项**：敏感数据的加密存储
- **数据清理**：彻底删除数据的功能

### 安全功能

- **扩展沙箱**：扩展在隔离环境中运行
- **权限控制**：精细的权限管理系统
- **安全审计**：安全事件日志记录
