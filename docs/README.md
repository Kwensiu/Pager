# Pager 用户文档

Pager 是一个基于 Electron + React + TypeScript 构建的桌面应用程序，提供侧边栏界面来管理和访问多个网站，旨在通过结构化组织网络资源来提高生产力。

## 快速开始

### 安装

1. 从 [GitHub Releases](https://github.com/Kwensiu/pager/releases) 下载最新版本
2. 运行安装程序
3. 启动 Pager 应用程序

### 首次使用

1. 启动后，您会看到一个侧边栏和主窗口
2. 点击侧边栏中的 "+" 按钮添加网站分组
3. 在分组中添加您常用的网站

## 核心功能

### 侧边栏管理

- **分类管理**：创建一级分类（如"工作"、"娱乐"）
- **分组管理**：在分类下创建二级分组（如"AI工具"、"开发工具"）
- **网站管理**：在分组中添加、编辑、删除网站

### 网站浏览

- **多标签页支持**：每个网站独立标签页
- **网站图标自动获取**：自动从网站获取 favicon
- **拖拽排序**：通过拖拽调整网站和分组顺序

### 界面定制

- **主题切换**：支持浅色、深色和跟随系统主题
- **侧边栏折叠**：可折叠侧边栏以节省空间
- **多语言支持**：支持中文和英文界面

## 使用指南

### 添加网站

1. 点击侧边栏中的 "+" 按钮
2. 选择"添加网站"
3. 输入网站名称和 URL
4. 点击"保存"

### 管理分组

1. 右键点击侧边栏空白处
2. 选择"添加分类"或"添加分组"
3. 输入分组名称
4. 将网站拖拽到分组中

### 编辑网站

1. 右键点击网站
2. 选择"编辑"
3. 修改网站信息
4. 点击"保存"

## 设置说明

### 外观设置

- **主题模式**：选择浅色、深色或跟随系统
- **界面语言**：选择中文或英文
- **侧边栏显示模式**：配置折叠时的显示方式

### 浏览器设置

- **启用 JavaScript**：允许网页执行 JavaScript
- **允许弹出窗口**：控制是否允许网页打开新窗口
- **保存会话数据**：保存登录信息和浏览历史

### 隐私与数据

- **退出时清除缓存**：关闭应用时清除临时文件
- **数据管理**：重置设置或清除所有数据

## 扩展管理

Pager 支持 Chrome 扩展，包括 Tampermonkey 脚本。

### 添加扩展

1. 打开"设置" → "扩展管理"
2. 点击"添加扩展"
3. 选择包含 `manifest.json` 的扩展文件夹或 ZIP 文件
4. 扩展将出现在管理列表中

### 扩展隔离

- **严格隔离**：完全隔离扩展，防止影响其他扩展
- **标准隔离**：适度隔离，平衡安全性和兼容性
- **宽松隔离**：最小隔离，最大化兼容性
- **无隔离**：不进行隔离（不推荐）

### 权限管理

- 查看扩展请求的权限
- 单独授权或拒绝每个权限
- 监控扩展的错误和活动

## 增强功能

### 浏览器指纹伪装

防止网站通过浏览器指纹进行追踪：

- **基础模式**：修改基本指纹信息
- **平衡模式**：修改更多指纹信息，平衡兼容性
- **高级模式**：全面修改指纹信息，最大隐私保护

### 全局快捷键

- **窗口置顶**：快速切换窗口置顶状态
- **小窗模式**：切换到迷你窗口模式
- **侧边栏切换**：显示/隐藏侧边栏

### 系统托盘

- 最小化到系统托盘
- 托盘图标快捷操作
- 系统通知

### 内存优化

- 自动清理不活跃的网站
- 配置清理间隔和最大不活跃时间
- 释放内存资源

### 数据同步

- 导出/导入配置
- 导出/导入 Cookie 数据
- 设置自动同步间隔

## 常见问题

### 网站图标不显示

1. 确保网站 URL 正确
2. 尝试右键点击网站 → "编辑" → "刷新图标"
3. 检查网络连接

### 扩展无法加载

1. 确保扩展格式正确（标准 Chrome 扩展）
2. 检查 `manifest.json` 文件是否存在且格式正确
3. 尝试重启应用程序

### 应用程序卡顿

1. 启用"内存优化"功能
2. 减少同时打开的网站数量
3. 检查系统资源使用情况

## 故障排除

### 应用程序无法启动

1. 尝试重新安装应用程序
2. 检查系统是否满足要求（Windows 10+，macOS 10.15+，Linux）
3. 查看日志文件（位于用户数据目录）

### 数据丢失

1. 定期使用"数据同步"功能备份配置
2. 避免手动删除应用程序数据目录
3. 重置设置前确认已备份重要数据

### 网络问题

1. 检查代理设置（"设置" → "增强功能" → "代理支持"）
2. 确保网络连接正常
3. 尝试禁用浏览器指纹伪装功能
