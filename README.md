# Pager

Pager 是一个基于 Electron + React + TypeScript 构建的桌面应用程序，提供侧边栏界面来管理和访问多个网站，旨在通过结构化组织网络资源来提高生产力。

**注意：本项目目前处于早期测试阶段，功能和特性可能在开发过程中发生重大变化。**

## 项目设置

### 安装依赖

```bash
$ yarn install
```

### 开发模式

```bash
$ yarn dev

$ yarn format
```

### 构建应用程序

```bash
# Windows
$ yarn build:win

# macOS
$ yarn build:mac

# Linux
$ yarn build:linux
```
